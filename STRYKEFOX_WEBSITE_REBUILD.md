## UX CLEANUP NOTES

### Remove
- All book references and teasers
- "Revolutionizing healthcare" language
- Founder trauma narratives
- Generic healthcare marketing clichés
- Emotional storytelling elements
- Publisher-facing language
- "Join the movement" CTAs

### Compress
- Founder bio to operator credentials only
- Platform descriptions to functional statements
- Capability explanations to bullet points
- Philosophy sections to numbered principles

### Reorder
- Platform-first messaging
- Architecture before founder story
- Capabilities organized by function
- CTA positioned for action, not emotion

### Add
- Compliance credentials prominently
- Regulatory certifications visible
- Process documentation emphasis
- Operational metrics where available
- Clear partnership pathways
